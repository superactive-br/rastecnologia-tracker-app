Política de Privacidade

Rastec Tracker - Rastec Tecnologia

1. Introdução

A Rastec Tecnologia ("nós", "nosso" ou "empresa") está comprometida em proteger a privacidade dos usuários do aplicativo Rastec Tracker ("aplicativo"). Esta Política de Privacidade descreve como coletamos, usamos, armazenamos e protegemos suas informações pessoais.

Ao utilizar o Rastec Tracker, você concorda com as práticas descritas nesta política.

2. Informações que Coletamos

O Rastec Tracker coleta os seguintes tipos de informações:

2.1. Dados de Localização

  • Localização GPS: Coletamos dados de localização em tempo real dos dispositivos rastreadores vinculados à sua conta para fornecer os serviços de rastreamento.
  • Histórico de Localização: Armazenamos o histórico de rotas e posições para permitir a geração de relatórios e análises.

2.2. Dados da Conta

  • Credenciais de Acesso: E-mail e senha para autenticação na plataforma.
  • Token de Autenticação: Armazenado de forma segura no dispositivo para manter sua sessão ativa.

2.3. Dados do Dispositivo

  • Token de Notificação Push: Identificador único para envio de notificações sobre eventos dos seus rastreadores.
  • Informações do Dispositivo: Modelo, sistema operacional e versão do aplicativo para fins de suporte técnico.

2.4. Dados de Uso

  • Analytics: Utilizamos o Firebase Analytics para entender como o aplicativo é utilizado e melhorar a experiência do usuário.
  • Relatórios de Erros: O Firebase Crashlytics coleta informações sobre falhas do aplicativo para correção de bugs.

3. Como Usamos suas Informações

Utilizamos as informações coletadas para:

  • Fornecer serviços de rastreamento GPS em tempo real
  • Enviar notificações sobre eventos dos seus dispositivos (alertas de velocidade, cercas virtuais, etc.)
  • Gerar relatórios e históricos de movimentação
  • Autenticar seu acesso à plataforma
  • Melhorar a qualidade e desempenho do aplicativo
  • Fornecer suporte técnico
  • Cumprir obrigações legais

4. Compartilhamento de Dados

Não vendemos, alugamos ou compartilhamos suas informações pessoais com terceiros para fins de marketing.

Podemos compartilhar dados apenas nas seguintes situações:

  • Provedores de Serviço: Utilizamos serviços do Google Firebase para analytics, notificações push e relatórios de erros. Esses serviços estão sujeitos às políticas de privacidade do Google.
  • Requisitos Legais: Quando exigido por lei, ordem judicial ou autoridade governamental competente.
  • Proteção de Direitos: Para proteger nossos direitos, privacidade, segurança ou propriedade.

5. Armazenamento e Segurança

Implementamos medidas de segurança técnicas e organizacionais para proteger suas informações:

  • Criptografia: Dados sensíveis são armazenados de forma criptografada no dispositivo.
  • Autenticação Biométrica: Opção de proteção adicional via Face ID ou impressão digital.
  • Conexões Seguras: Toda comunicação com nossos servidores utiliza protocolo HTTPS.
  • Servidores Seguros: Nossos servidores são protegidos com as melhores práticas de segurança da indústria.
Retenção de Dados: Mantemos seus dados enquanto sua conta estiver ativa ou conforme necessário para fornecer nossos serviços. Você pode solicitar a exclusão dos seus dados a qualquer momento.

6. Seus Direitos

De acordo com a Lei Geral de Proteção de Dados (LGPD), você tem direito a:

  • Acessar seus dados pessoais
  • Corrigir dados incompletos ou desatualizados
  • Solicitar a exclusão dos seus dados
  • Revogar o consentimento para uso dos dados
  • Solicitar a portabilidade dos dados
  • Obter informações sobre o compartilhamento dos dados

Para exercer esses direitos, entre em contato conosco através dos canais informados abaixo.

7. Permissões do Aplicativo

O Rastec Tracker solicita as seguintes permissões:

  • Localização: Para exibir sua posição no mapa e calcular distâncias.
  • Câmera: Para leitura de QR codes de configuração de dispositivos.
  • Notificações: Para enviar alertas sobre eventos dos seus rastreadores.
  • Internet: Para comunicação com os servidores de rastreamento.
  • Biometria: Para autenticação segura opcional.

8. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Notificaremos sobre mudanças significativas através do aplicativo ou por e-mail. Recomendamos revisar esta política regularmente.
